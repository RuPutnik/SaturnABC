Общее руководство пользования программой SaturnAB версия 1.0
Данная программа имеет осоновную функцию в виде шифрования текста
различными способами. Автор программы Илья Румянцев (Putnik's production)

--Основное окно--
В основном окне доступны два больших поля для ввода текста. Левое - для текста незашифрованного.
В правое поле текст поступает после шифрации. Также возможна обратная ситуация- в правое поле поступает 
шифрованный текст, а в левом получается дешифврованный. Для шифрования и дешифрования необходимо 
нажать на соответсвующую кнопку под двумя главными текстовыми полями.
Четыре нижние кнопки в левой стороне экрана позволяют очистить оба текстовых поля и поле для ключа.
Поле для ключа(центр экрана) Используется для ввода ключей шифрования всех видов. Случайное 
значение ключа может быть сгенерировано специальной кнопкой "Случайный", расположенной рядом с 
полем для ключа.
Меню основного окна позовляет:
1)Открыть окно выбора типа шифрования
2)Открыть настройки)
3)Сохранить шифрованный и нешифрованный текст(текст из правых и левых окон) или открыть текст из файла
4)Открыть справку
5)Выйти из программы

--Окно выбора ключа--
Данное окно позовляет выбирать алгоритм шифрования из нескольких достпуных или 
же подключить свой модуль шифрования.
Изначально доступны 5 видов:
1)Шифр Цезаря
2)Шифр кодовым словом
3)Блочный шифр
4)Шифр Тритемиуса
5)Расширяющий шифр
Подробнее о каждом из алгоритмов можно прочитать в 
текстовом окне в левой половине экрана при нажатии
на соотвествующий тип в правом списке.

--Настройки--
Программа позволяет устанавливать 3 вида параметров:
1)Шифрование или не шифрование пробела
2)Внутреннее системное логгирование в файла
3)Выбор пользовательской таблицы индексов символов
Последняя опция позволяет вручную указывать изначальным индекс для каждого добавленного в нее символа. 
Данный индекс будет использоваться для шифрования соответствующего символа.
